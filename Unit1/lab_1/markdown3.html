Markdown

What it is?

Markdown is a maca language that makes it easy to apply formatting to text employing a series of features of a special format, it was developed by John Gruber with the help of Aaron Swartz in 2004 in order to that people could write using a text format plain easy to read, easy to write and with the possibility of converting your valid HTML document.

Basic Syntax

Bold text

Image ![Text] (Imf.png).

Example:

Var foo = `bar´;

Advantages:

  1. It is more difficult to make syntax errors
  2. Writing for the web is faster and more comfortable

Disadvantage:

  1. we must memorize the function of the characters and the order they should be in.

  2. It is not as powerful as other languages

why use it?

  • It is compatible with all platforms
  • See the difference between a file created by a word processor and the same file opened with a plain text editor

Difference between Markdown and HTML

HTML is a publishing format; Markdown is a writing format. Additionally, Markdown's formatting syntax only targets issues that can be expressed in plain text. For those markups that are not covered by Markdown syntax, simply use the HTML itself.
